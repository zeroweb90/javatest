Quickstart: Facebook SDK for JavaScript

FaceBook SDK يوفر لك مجموعة من الوظائف التي:

  • تتيح لك إستخدام like button و العديد من social plugins في موقعك.
  • تتيح لك إستخدام Facebook login لتسهيل تسجيل عملائك في موقعك.
  • تسهل لك الإتصال ب Facebook graph Api.
  • تسهل لك التواصل عندما تنشأ لعبة أو صفحة بالFacebook خاصة بموقعك.

المتصفحات المتاحة

Facebook SDK يدعم أخر إصدارين من أغلب المتصفحات المشهورة :chrome, firefox, Edge, Safari(ios) and intrnet explorer (إصدار 11 فقط)

المتصفحات المتاحة

Facebook SDK لا يحتاج لتحميل أو تثبيت أي نوع ملفات, فقط تحتاج لنقل الJs code لصفحة الـ html الخاصة بك.

* يجب وضع الcode بعد فتح عنصر الـ'body' مباشرة لمنع حدوث أي تعطيل من خلال عناصر الcode الخاصة بك.

<script>
  window.fbAsyncInit = function() {
    FB.init({
      appId            : 'your-app-id',
      autoLogAppEvents : true,
      xfbml            : true,
      version          : 'v3.2'
    });
  };

  (function(d, s, id){
     var js, fjs = d.getElementsByTagName(s)[0];
     if (d.getElementById(id)) {return;}
     js = d.createElement(s); js.id = id;
     js.src = "https://connect.facebook.net/en_US/sdk.js";
     fjs.parentNode.insertBefore(js, fjs);
   }(document, 'script', 'facebook-jssdk'));
</script>

هذا الcode سوف يقوم بتشغيل الSDK . يجب عليك الان نسخ الـ Facebook ID الخاص بك في خانة your-app-id , يمكنك جلبه من هنا https://developers.facebook.com/apps
